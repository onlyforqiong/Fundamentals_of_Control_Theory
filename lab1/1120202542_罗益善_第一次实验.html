第一次实验

实验目的

1. 掌握利用 MATLAB 建立控制系统模型的方法。
2. 掌握系统的各种模型表述及相互之间的转换关系。
3. 学习和掌握系统模型连接的等效变换

实验原理

  • 系统的模型描述了系统的输入、输出变量以及内部各变量之间的关系，表征一个系统的模型有很多种，如微分方程、传递函数、状态空间方程等。这里主要介绍系统多项式型传递函数（TF）模型、零极点型传递函数（ZPK）模型和状态空间方程（SS）模型

    传递函数（TF）模型

    传递函数是描述线性定常系统输入-输出关系的一种最常用的数学模型，其表达式一般为G(s)=bmsm+bm−1sm−1++b1s1+b0ansn+an−1sn−1++a1s1+a0G(s) = \frac{b_ms^m +b_{m-1}s^{m-1}+ +b_1s1+b_0}{a_ns^n +a_{n-1}s^{n-1}+ +a_1s^1+a_0}G(s)=an​sn+an−1​sn−1++a1​s1+a0​bm​sm+bm−1​sm−1++b1​s1+b0​​
    在 MATLAB 中，直接使用行向量分子分母多项式的表示系统，即
    num=[bm,bm−1,…b1,b0]num = [bm, bm-1, … b1, b0]num=[bm,bm−1,…b1,b0]
    den=[an,an−1,…a1,a0]den = [an, an-1, … a1, a0]den=[an,an−1,…a1,a0]
    调用 tf 函数可以建立传递函数 TF 对象模型，调用格式如下：
    Gtf=tf(num,den)Gtf = tf(num,den)Gtf=tf(num,den)
    Tfdata 函数可以从 TF 对象模型中提取分子分母多项式，调用格式如下：
    [num,den]=tfdata(Gtf)[num,den] = tfdata(Gtf)[num,den]=tfdata(Gtf)返回 cell 类型的分子分母多项式系数
    [num,den]=tfdata(Gtf,′v′)[num,den] = tfdata(Gtf,'v')[num,den]=tfdata(Gtf,′v′) 返回向量形式的分子分母多项式系数

    零极点增益（ZPK）模型

    传递函数因式分解后可以写成
    G(s)=k(s−z1)(s−z2)...(s−zm)(s−p1)(s−p2)...(s−pn)G(s) = \frac{k(s-z_1)(s-z_2)...(s-zm)}{(s-p_1)(s-p_2)...(s-p_n)}G(s)=(s−p1​)(s−p2​)...(s−pn​)k(s−z1​)(s−z2​)...(s−zm)​
    式中， z1, z2 ,… , zm 称为传递函数的零点， p1, p2 ,… , pn 称为传递函数的极点，k为传递系数(系统增益).在 MATLAB 中，直接用[z,p,k]矢量组表示系统，其中 z，p，k 分别表示系统的零极
    点及其增益，即：
    z=[z1,z2,…,zm];p=[p1,p2,…,pn];k=[k];z=[z1,z2,…,zm]; p=[p1,p2,…,pn]; k=[k];z=[z1,z2,…,zm];p=[p1,p2,…,pn];k=[k];
    调用 zpk 函数可以创建 ZPK 对象模型，调用格式如下：
    Gzpk=zpk(z,p,k)Gzpk = zpk(z,p,k)Gzpk=zpk(z,p,k)
    同样，MATLAB 提供了 zpkdata 命令用来提取系统的零极点及其增益，调用格式如下：
    [z,p,k]=zpkdata(Gzpk)[z,p,k] = zpkdata(Gzpk)[z,p,k]=zpkdata(Gzpk) 返回 cell 类型的零极点及增益
    [z,p,k]=zpkdata(Gzpk,’v’)[z,p,k] = zpkdata (Gzpk,’v’)[z,p,k]=zpkdata(Gzpk,’v’) 返回向量形式的零极点及增益
    函数 pzmap 可用于求取系统的零极点或绘制系统的零极点图，调用格式如下：
    pzmap(G)pzmap(G)pzmap(G) 在复平面内绘出系统模型的零极点图。
    [p,z]=pzmap(G)[p,z] = pzmap(G)[p,z]=pzmap(G) 返回的系统零极点，不作图。

    状态空间（SS）模型

    由状态变量描述的系统模型称为状态空间模型，由状态方程和输出方程组成：{x˙=Ax+Buy=Cx+Du\left\{\begin{aligned}\dot{x}&= Ax + Bu\\y&=Cx + Du\end{aligned}\right.{x˙y​=Ax+Bu=Cx+Du​
    其中：x 为 n 维状态向量；u 为 r 维输入向量； y 为 m 维输出向量； A 为 n×n 方阵，称为
    系统矩阵；B 为 n×r 矩阵，称为输入矩阵或控制矩阵；C 为 m×n 矩阵，称为输出矩阵；D
    为 m×r 矩阵，称为直接传输矩阵。
    在 MATLAB 中，直接用矩阵组[A,B,C,D][A,B,C,D][A,B,C,D]表示系统，调用 ss 函数可以创建 SS 对象模型，调用格式如下：
    Gss=ss(A,B,C,D)Gss = ss(A,B,C,D)Gss=ss(A,B,C,D)
    同样，MATLAB 提供了 ssdata 命令用来提取系统的 A、B、C、D 矩阵，调用格式如下：
    [A,B,C,D]=ssdata(Gss)[A,B,C,D] = ssdata (Gss)[A,B,C,D]=ssdata(Gss)返回系统模型的 A、B、C、D 矩阵

    三种模型之间的转换

    上述三种模型之间可以互相转换，MATLAB 实现方法如下
    • TF 模型→ZPK 模型：zpk(SYS)或tf2zp(num,den)zpk(SYS)或 tf2zp(num,den)zpk(SYS)或tf2zp(num,den)
    • TF 模型→SS 模型：ss(SYS)或tf2ss(num,den)ss(SYS)或 tf2ss(num,den)ss(SYS)或tf2ss(num,den)
    • ZPK 模型→TF 模型：tf(SYS)或zp2tf(z,p,k)tf(SYS)或 zp2tf(z,p,k)tf(SYS)或zp2tf(z,p,k)
    • ZPK 模型→SS 模型：ss(SYS)或zp2ss(z,p,k)ss(SYS)或 zp2ss(z,p,k)ss(SYS)或zp2ss(z,p,k)
    • SS 模型→TF 模型：tf(SYS)或ss2tf(A,B,C,D)tf(SYS)或 ss2tf(A,B,C,D)tf(SYS)或ss2tf(A,B,C,D)
    • SS 模型→ZPK 模型：zpk(SYS)或ss2zp(A,B,C,D)zpk(SYS)或 ss2zp(A,B,C,D)zpk(SYS)或ss2zp(A,B,C,D)

分析

  • 在这种情况下，反馈系统的闭环传递函数是一个复杂的多项式，其中包含多个因子。这些因子描述了系统的不同特性，例如极点和零点。极点是使系统不稳定的因素，而零点是使系统更稳定的因素。
  • 在这种情况下，反馈系统的闭环传递函数包含两个极点和三个零点。这些因子可以用来分析和设计线性控制系统。
